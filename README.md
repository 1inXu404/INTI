# INTI — Insta Nexus Type Indicator

> 测测你是哪种影石玩家 🎯

一款为 **影石 Insta360** 设计的趣味人格测试 H5 页面，通过 10 道题测出你最适合哪款影石产品。

## ✨ 特性

- 🎲 **随机出题**：5 维度 × 2 题 = 10 题，从 40 题题库随机抽取，每次测试体验不同
- 🧠 **模板匹配**：基于 L/M/H 三级评分的模板距离算法，科学匹配 9 种人格
- 🛒 **购买直达**：结果页集成产品购买链接，一键跳转官方商城
- 📱 **移动端优先**：适配手机屏幕，分享到社交平台体验最佳
- 🖼️ **单文件可迁移**：产品图 base64 内嵌，部署零依赖

## 🧩 测试维度

| 维度 | 说明 |
|------|------|
| 创作场景 | 你拍什么、在哪拍、拍的频率 |
| 便携执念 | 对设备重量和体积的敏感程度 |
| 画质洁癖 | 对分辨率、帧率、传感器尺寸的追求 |
| 技术信仰 | 对新功能、AI 能力、参数规格的热衷程度 |
| 表达风格 | 你剪视频的方式和内容输出风格 |

## 🏗️ 技术架构

```
├── index.html              # 纯 DOM 结构
├── src/
│   ├── data/
│   │   ├── quiz-data.js    # 维度定义、题库、人格模板、匹配规则
│   │   └── posters.js      # 产品海报（base64）
│   ├── scripts/
│   │   └── app.js          # 核心逻辑：渲染、计分、匹配、结果
│   └── styles/
│       └── main.css        # 样式
└── images/                 # 原始产品图
```

**技术栈**：纯前端 ES Module，无构建工具，无后端依赖。

**匹配算法**：每维度 2 题原始分 → 映射为 L/M/H → 与 9 个人格模板计算欧氏距离 → 最近匹配。

## 🚀 本地运行

```bash
# 需要 HTTP 服务（ES Module 限制，不能 file:// 打开）
cd insta360-personality-test
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 📦 部署到 GitHub Pages

1. Fork 或克隆本仓库
2. 仓库 → **Settings** → **Pages**
3. Source 选 **Deploy from a branch**，Branch 选 `main`，目录 `/ (root)`
4. 等待 1-2 分钟，访问 `https://<用户名>.github.io/<仓库名>/`
