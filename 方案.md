# 影石 InstaType 人格测试方案

## 一、测试概述

**测试名称**：INTI（Insta Nexus Type Indicator）影石人格测试  
**副标题**：测测你是哪种影像玩家  
**题量**：25题（5维度 × 5题）  
**测试时长**：约2-3分钟  
**人格类型数**：9种  
**风格定位**：轻松有趣，带点自嘲，贴合年轻人表达习惯  

---

## 二、测试维度

### 维度1：创作场景（Where do you shoot?）
衡量用户主要在什么场景下使用相机。

| 极值 | 描述 |
|------|------|
| 🏔️ 冒险家 | 极限运动、户外探险、旅行远征 |
| 🏙️ 城市猎人 | 日常通勤、城市漫步、美食探店 |
| 🎬 创作者 | 有策划的内容创作、Vlog、短视频 |
| 🛋️ 宅家记录 | 家庭生活、宠物、孩子、日常碎片 |

### 维度2：便携执念（How light is your load?）
衡量对设备体积/重量的敏感度。

| 极值 | 描述 |
|------|------|
| 🪶 极简主义者 | 越小越好，口袋必须有空 |
| 🎒 实用主义 | 够用就行，但别太重 |
| 🎒🎒 装备党 | 重不怕，功能全就行 |

### 维度3：画质洁癖（How much do you care?）
衡量对画质的要求程度。

| 极值 | 描述 |
|------|------|
| 🔍 像素级洁癖 | 8K起步，糊了就删 |
| 📱 清晰就够 | 发社交媒体够用就行 |
| 🎲 随缘派 | 能记录下来就是胜利 |

### 维度4：技术信仰（How much AI is too much?）
衡量对新技术/AI功能的态度。

| 极值 | 描述 |
|------|------|
| 🤖 科技狂热 | AI功能越多越好，手动是耻辱 |
| ⚖️ 理性尝鲜 | 新技术可以试，但别替我做决定 |
| 📻 怀旧派 | 别给我整花活，能拍就行 |

### 维度5：表达风格（How do you share?）
衡量用户记录和分享内容的风格。

| 极值 | 描述 |
|------|------|
| 🎭 戏剧派 | 花式剪辑、特效拉满、必须炸裂 |
| 📖 叙事派 | 讲故事、有节奏、有情绪 |
| 📸 直出派 | 拍完就发，后期是什么？ |
| 🔒 记录派 | 主要给自己看，不太分享 |

## 四、题目设计（25题）

### 维度1：创作场景（Q1-Q5）

**Q1** 周末你最可能在干嘛？
- A. 在山上/海里/天上作死（冒险家）
- B. 在商场/咖啡店/网红点打卡（城市猎人）
- C. 在研究怎么剪出爆款（创作者）
- D. 在家陪猫/陪娃/陪沙发（宅家记录）

**Q2** 你手机相册里最多的是什么？
- A. 风景大片，随便一张都能当壁纸（冒险家）
- B. 美食和探店照，相册像个点评网站（城市猎人）
- C. 素材素材素材，剪辑的时候再挑（创作者）
- D. 家人/宠物的丑照，删都删不完（宅家记录）

**Q3** 你最想记录的瞬间是？
- A. 跳伞开伞那3秒，肾上腺素拉满（冒险家）
- B. 偶遇的宝藏小店，发出来肯定很多人问地址（城市猎人）
- C. 我精心策划的Vlog开场，必须一镜到底（创作者）
- D. 孩子第一次走路/猫第一次跳上桌子（宅家记录）

**Q4** 朋友约你出去玩，你的第一反应是？
- A. 有没有刺激的？没挑战不去（冒险家）
- B. 有什么好吃的/好拍的？发我定位（城市猎人）
- C. 这次拍什么主题？我带设备（创作者）
- D. 带上家人一起吧，人多热闹（宅家记录）

**Q5** 如果相机丢了，你最心疼的是？
- A. 刚才爬山/冲浪拍的素材，删了就没了（冒险家）
- B. 探店拍的美食视频，那家店已经倒闭了（城市猎人）
- C. 熬夜剪了一半的片子，工程文件还没备份（创作者）
- D. 孩子/宠物的成长视频，钱可以再赚（宅家记录）

### 维度2：便携执念（Q6-Q10）

**Q6** 你的背包/口袋通常？
- A. 空的，能不带就不带，身上越轻越好（极简主义者）
- B. 刚好装下手机钥匙纸巾，再多就沉了（实用主义）
- C. 像个移动军火库，什么都有（装备党）

**Q7** 买包/买衣服时你会考虑？
- A. 有没有口袋能塞下相机（极简主义者）
- B. 好看就行，相机另外拿手上（实用主义）
- C. 能挂多少配件，口袋越多越好（装备党）

**Q8** 出门旅行你带几台设备？
- A. 手机+一台小相机，够了（极简主义者）
- B. 手机+相机+充电宝，标配（实用主义）
- C. 手机+相机+稳定器+三脚架+备用电池+滤镜...（装备党）

**Q9** 你对"口袋相机"的态度是？
- A. 真正的口袋！塞进去就走，碰不到才算 pocket（极简主义者）
- B. 口袋...勉强能放，鼓一点也行（实用主义）
- C. 口袋？我有专门的相机包，专业！（装备党）

**Q10** 如果一台相机画质顶级但很重，你会？
- A. 不考虑，轻才是正义，画质够用就行（极简主义者）
- B. 看情况，重一点也能接受，别太离谱（实用主义）
- C. 画质好就行，重不重无所谓，我有麒麟臂（装备党）

## 五、计分逻辑

### 维度计分
每个维度独立计分，统计各选项出现次数最多的类别。

### 产品匹配优先级

**主维度权重**：
- 创作场景 → 决定产品大类（全景/运动/口袋/稳定器）
- 便携执念 → 决定同类别中的轻量/旗舰款
- 画质洁癖 → 决定入门/旗舰款
- 技术信仰 → 决定是否推荐AI功能强的产品
- 表达风格 → 辅助推荐配件/玩法

### 匹配决策树

```
创作场景
├── 冒险家
│   ├── 便携=极简 → X4 Air
│   ├── 便携=装备党 + 画质=洁癖 → X5
│   └── 便携=装备党 + 技术=狂热 → Ace Pro 2
├── 城市猎人
│   ├── 表达=直出/记录 → GO Ultra
│   ├── 表达=戏剧/叙事 → X4 Air
│   └── 自拍需求高 → Snap
├── 创作者
│   ├── 画质=洁癖 → X5
│   ├── 表达=戏剧 → Flow 2 Pro + Ace Pro 2
│   └── 便携=极简 → GO Ultra
└── 宅家记录
    ├── 便携=极简 → GO 3S
    └── 便携=实用 → GO Ultra
```
